## 🚀 Getting Started

### 🖥️ Frontend

1. Open a terminal and navigate to the frontend folder, then install dependencies and start the dev server:

```bash
cd app
npm install
npm run dev
```

The React/Tailwind UI will be served at **http://localhost:5173** by default.

---

### 🧠 Backend

Change directory to the backend and install dependencies:

```bash
cd ..
cd backend
pip install -r requirements.txt
python app.py
```
